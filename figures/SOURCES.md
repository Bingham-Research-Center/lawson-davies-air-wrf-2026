# Figure provenance

Every manuscript figure should be reproducible. Record here, per figure file, where it came from:
the generating script (repo · path · commit, usually on CHPC in `brc-wrf` / `brc-tools`), the input
data or WRF run, and any manual steps. Keep this current as figures land in `figures/`.

| Figure file | Script (repo · path · commit) | Data / WRF run | Notes |
|-------------|-------------------------------|----------------|-------|
